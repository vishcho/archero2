> 依據 [2026-07-03-round2-matchup.md](./2026-07-03-round2-matchup.md) 對陣表整理的下注建議。
> 每組下注 7 場：R1(4) → R2(2) → 組決賽(1)。
> **R2 對戰：A/C 勝者互打、B/D 勝者互打**（已確認）。

## 判讀原則

依可信度排序（**淘汰賽成績 > 戰力 > 資格賽時間**）：

1. **上屆淘汰賽最佳通關時間** — 真刀真槍打出來的，最可信。有紀錄者優先於無紀錄者；同有紀錄者比時間。
2. **本屆戰力** — 客觀數值，差距大（>25%）時可直接定勝負。
3. **本屆資格賽通關時間** — **帶有水分**（全員通關、可刷可摸魚），僅在前兩項無法區分時當弱參考。

信心標記：

| 標記 | 意義 | 建議注量 |
| ---- | ------------------ | -------- |
| ✅   | 穩（數據全面壓制） | 重壓     |
| ⚖    | 微傾（有依據但有變數） | 中注 |
| 🎲   | 五五波（純博）     | 輕注或看賠率 |
