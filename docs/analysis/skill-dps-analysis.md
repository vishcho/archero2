## 關鍵發現

### 1. 蓄力巨箭是負收益技能（毒技能）

蓄能環＋蓄能射線原本 8.99B/s，加入蓄力巨箭後崩到 3.18B/s（**-65%**），
幾乎退回蓄能環單獨的水準——蓄能射線的貢獻被整個抵銷。
推測它改變蓄力機制（拉長蓄力時間或改為單發巨箭），
導致蓄能射線觸發頻率暴跌。**在蓄能流中絕對避開。**

它也會壓掉其他協同：五技能組（含巨箭）後段 DPS ≈
「被巨箭拖垮的蓄力組（3.18）＋追蹤之眼單體（1.24）」的純加總，
連追蹤之眼的放大效果都被消掉。

### 2. 追蹤之眼是放大器，不是輸出技能

單獨只有 1.24B/s（全場最弱），但疊上蓄能環實測 6.17B/s，
比純加總（4.21）高 **47%**。單體強度和疊加價值完全是兩回事。

### 3. 輸出在第四技能後飽和

第五格疊雙箭僅 +1.8%。**第五格建議拿生存技能（如 MAX 愛心），輸出損失無感。**

### 4. 實測與 archero2hub tier list 高度吻合

增量前三正是該站三個 Tier 1 技能；雙箭（Tier 5）實測 ≈0。
唯一超出 tier list 的情報：能量箭（蓄力巨箭）不只是「弱」（Tier 5），
在蓄能流中是**主動負面**。

## 結論

- **最佳輸出組合：蓄能環＋追蹤之眼＋蓄能射線＋連射**（15.54B/s，基準的 5.2 倍）
- 選技能優先序：**蓄能射線 > 連射 ≈ 追蹤之眼 >（蓄能環基底）>> 光翼重傷箭 > 雙箭 >>> 蓄力巨箭（負收益）**
- 第五格拿生存技能
