# 明星杯規則

## 賽制流程

| 階段       | 說明                                                                  |
| ---------- | --------------------------------------------------------------------- |
| **資格賽** | 全員參加，排行榜前 **64 名**晉級淘汰賽；排名依據：通關層數 + 完成時間 |
| **淘汰賽** | 64 人分為 **8 組**，每組 8 人對戰；每組**第一名**晉級決賽             |
| **決賽**   | 8 名晉級者**兩兩對決**，最終勝者奪得冠軍                              |

## 競猜功能（淘汰賽 & 決賽適用）

| 結果     | 獎勵                                      |
| -------- | ----------------------------------------- |
| 競猜成功 | 獲得投注競猜幣 **×1.5** 的明星徽章        |
| 競猜失敗 | 獲得投注競猜幣 **×0.5**（一半）的明星徽章 |

> 下注所需競猜幣金額會依賽事階段調整。

## 注意事項

- 賽事有時間限制，**未按時參賽視為棄權**。

---

## 資格賽排名規則（優先順序）

| 優先級 | 判定條件                   | 說明                         |
| ------ | -------------------------- | ---------------------------- |
| 1      | **最高關卡進度**           | 進度越高排名越前             |
| 2      | **對最終 BOSS 造成的傷害** | 擊殺數相同時，傷害較高者優先 |
| 3      | **通關時間**               | 傷害相同時，時間較短者優先   |
| 4      | **達成先後順序**           | 以上全部相同時，先達成者排前 |

- 資格賽排行榜僅用於晉級判定，**不發放排行榜獎勵**
- 僅顯示前 **1000 名**

## 淘汰賽排名規則

| 名次     | 條件                                                         |
| -------- | ------------------------------------------------------------ |
| 第 1 名  | 冠軍                                                         |
| 第 2 名  | 亞軍                                                         |
| 第 3 名  | 季軍賽勝者                                                   |
| 第 4 名  | 季軍賽敗者                                                   |
| 其餘名次 | 依淘汰賽**勝場數**排名；勝場相同則依**資格賽成績**較佳者優先 |

> 排名獎勵於決賽結束後透過**郵件**發放。
